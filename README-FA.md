# 🇩🇪 ربات تلگرام آموزش زبان آلمانی

ربات هوشمند آموزش زبان آلمانی برای تلگرام، ساخته‌شده با Cloudflare Workers.

این ربات به فارسی‌زبانان کمک می‌کند تا زبان آلمانی را از طریق تصحیح گرامر، یادگیری واژگان، تمرین‌های آموزشی، آزمون‌های مبتنی بر جزوه و توضیحات هوش مصنوعی یاد بگیرند.

## امکانات

* ✍️ تصحیح گرامر و املای جملات آلمانی (`/check`)
* 📚 آموزش واژگان جدید همراه با مثال (`/vocab`)
* 🧠 توضیح قواعد گرامری با هوش مصنوعی (`/explain`)
* 📝 تولید تمرین همراه با پاسخ (`/practice`)
* 📄 ساخت سوال از روی عکس و فایل PDF جلسات (`/lesson`)
* 🔄 ارسال خودکار واژگان روزانه
* ⚙️ تنظیم تعداد کلمات و ساعت ارسال
* ☁️ اجرای کامل روی Cloudflare Workers
* 🤖 استفاده از Google Gemini و Cloudflare Workers AI

## دستورات

| دستور       | توضیح                                 |
| ----------- | ------------------------------------- |
| `/check`    | بررسی و اصلاح جمله آلمانی             |
| `/explain`  | توضیح گرامر یا واژگان                 |
| `/practice` | تولید تمرین آموزشی                    |
| `/vocab`    | دریافت واژگان جدید                    |
| `/lesson`   | ساخت تمرین از روی جزوه یا فایل آموزشی |
| `/auto on`  | فعال کردن ارسال روزانه                |
| `/auto off` | غیرفعال کردن ارسال روزانه             |
| `/settings` | نمایش تنظیمات فعلی                    |

## ساخته شده با

* Cloudflare Workers
* Cloudflare KV
* Telegram Bot API
* Google Gemini 2.5 Flash
* Cloudflare Workers AI (Llama 3.3)

## مناسب برای

* زبان‌آموزان آلمانی
* دانشجویان و دانش‌آموزان فارسی‌زبان
* یادگیری خودآموز
* مرور روزانه واژگان و گرامر
